## Setup
This project can be run on [Google Colab](https://colab.research.google.com).  
Required libraries:  
- Python 3.x  
- pandas  
- scikit-learn  
- matplotlib  

## How to Run
1. Open the notebook in Google Colab.  
2. Upload your dataset into the `data/` folder.  
3. Run the cells step by step.
